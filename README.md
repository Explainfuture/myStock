# Stockpedia

Stockpedia is an A-share (CN) investing learning site:

- Encyclopedia-style content (chapters -> articles)
- Split view for patterns (text + sticky chart)
- TradingView Lightweight Charts v5 (CN red-up / green-down)

## Dev

```bash
pnpm install
pnpm dev
```

## Environment

Copy `.env.example` to `.env.local` and fill values if you want Supabase-backed data.
